# Linked List

Linked list exemple for my website tutorial. Pretty simple.
Feel free to use it.

## How to use - *Unix based systems project*

### Compilation

Compile the program with ```make```.

Clean the program with ```make clean```.

### Utilisation

Lauch the binary as ```./bin/linked_list.out```.
